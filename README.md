# Dron Love

Test project for Python drone library integration.

## Description

This project contains test scripts for working with drone libraries in Python. It includes examples for basic drone control and real drone operations.

## Features

- Python drone library testing
- Basic drone control examples
- Real drone operation scripts

## Files

- `hello_drone.py` - Basic drone hello world example
- `real_drone.py` - Real drone control script

## Requirements

- Python 3
- Drone library (install according to your drone model)

## Usage

### Basic Example

```bash
python hello_drone.py
```

### Real Drone Control

```bash
python real_drone.py
```

## Note

This is a test project. Make sure to configure the appropriate drone library and connection settings before running with a real drone.
